Typography

Types Of Font

Heading Font: Noto Serif

I chose Noto Serif, which is a type of serif font, because it gives the web page a professional look. screen.

Paragraph Font: Lato

I chose Lato, which is a san serif type font, because it is easy to read and has a more modern look and feel to it.
